# Mangalam HDPE Pipes — Product Landing Page

A premium, responsive product landing page for **Mangalam HDPE Pipes**, built with vanilla HTML, CSS, and JavaScript.(Project by Gushwork)

## Live Preview

Open `index.html` directly in a browser — no build tools required.

## Project Structure

```
├── index.html          # Main HTML page
├── styles.css          # Complete stylesheet (tokens, components, responsive)
├── script.js           # Interactive features (carousel, tabs, modals, forms)
├── images/             # All image assets
│   ├── logo.png        # Brand logo
│   ├── BIS.svg         # BIS certification badge
│   ├── ISO.svg         # ISO certification badge
│   ├── CE.svg          # CE certification badge
│   ├── Product-img*.   # Product carousel images
│   ├── euroflex.png    # Trust logo
│   ├── raw-materials.jpeg
│   └── section-end/    # Portfolio section images
└── README.md
```

## Features

- **Product Image Carousel** — Swipeable slides with thumbnail navigation and hover zoom preview
- **Sticky Announcement Banner** — Appears on scroll with a call-to-action
- **Interactive Process Tabs** — Step-by-step manufacturing process walkthrough
- **Applications Carousel** — Horizontally scrollable industry application cards
- **FAQ Accordion** — Expandable/collapsible question panels
- **Contact & Quote Modals** — Form validation with success feedback
- **Catalogue Request** — Inline email capture form
- **Scroll Reveal Animations** — Fade-in on scroll using Intersection Observer
- **Mobile Responsive** — Hamburger menu, stacked layouts, touch-friendly

## Tech Stack

| Layer   | Technology                        |
|---------|-----------------------------------|
| HTML    | Semantic HTML5                    |
| CSS     | Vanilla CSS with custom properties|
| JS      | Vanilla ES6+ (no frameworks)     |
| Fonts   | Google Fonts (DM Sans, Playfair Display) |

| Breakpoint | Target          |
|-----------|-----------------|

| ≤ 800px   | Mobile (hamburger menu) |
| ≤ 480px   | Small phones    |
| ≤ 360px   | Compact phones  |
